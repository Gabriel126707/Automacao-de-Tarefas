# Automacao-de-Tarefas
Eu desenvolvi minha primeira automação utilizando python e a biblioteca pyautogui, essa automação me ajuda a fazer relatórios de vendas diários, onde eu fico com a responsabilidade de verificar os valores de vendas e a automação com a responsabilidade de extrair, tratar e colar as informações de venda do dia, na planilha principal de Vendas.
